# 🌐 Acerca del proyecto

**Web3 Rental Dashboard** es una plataforma empresarial Web 3.0 diseñada para monitorear inversiones, retornos financieros, eventos, artículos y contratos de renta mediante dashboards inteligentes y herramientas analíticas modernas.

El sistema fue desarrollado para:

- 💰 Monitorear inversiones y retornos
- 📊 Visualizar métricas en tiempo real
- 🎉 Gestionar eventos y contratos
- 🛒 Administrar artículos rentados
- 📈 Analizar ingresos y operaciones
- 🌐 Integrar tecnologías Web3
- ⚡ Automatizar procesos empresariales

La plataforma combina interfaces modernas, dashboards interactivos y arquitectura escalable orientada a soluciones descentralizadas.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaGestionAlquileresweb3.0.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaGestionAlquileresweb3.0
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Configurar variables de entorno

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
BLOCKCHAIN_RPC=
```

---

## 5️⃣ Ejecutar servidor

```bash
npm start
```

---

## 6️⃣ Abrir aplicación

```bash
http://localhost:3000
```
